Experiência Profissional

Apesar de nunca ter ocupado um cargo de programação, tive uma experiência que julgo interessante. Em 2016 eu começava a trabalhar como professor numa Escola Família Agrícola (EFA). Fui coordenador, professor e tinha sob minha tutoria um grupo de 18 alunos. Foram dias incríveis de grandes aprendizados, naturalmente desenvolvi habilidades como inteligência emocional, pensamento crítico para resoluções de problemas, e aprendi o valor e a importância de se trabalhar numa equipe conectada. Hoje, passados cinco anos, estou determinado a trabalhar como desenvolvedor Front-end da web e acredito que poderei seguir compartilhando conhecimento.

Tecnologias

HTML

É a linguagem de marcação padrão para a criação de páginas da Web, além do conteúdo, tem uma parte semântica importante para os mecanismos de busca.

CSS

É a linguagem responsável por estilizar a página web e especificar o layout. O CSS descreve como os elementos HTML devem ser exibidos.

Design

É o que dá palatividade ao projeto. Existem técnicas bem definidas que vai desde o layouts, passando pelo conteúdo até a melhor cor a ser aplicada nas páginas.

JavaScript

O JavaScript cuida das interações com o usuário e programa o comportamento das páginas da web.

Git

O sistema de controle de versão distribuído, o git, permite a manutenção do código e o trabalho em equipe.

Wordpress

O wordpress facilita a criação de sites de conteúdo dinâmicos ou loja virtual com o Woocommerce.
