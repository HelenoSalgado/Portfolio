Tecnologias

HTML

É a linguagem de marcação padrão para a criação de páginas da Web, além do conteúdo, tem uma parte semântica importante para os mecanismos de busca.

CSS

É a linguagem responsável por estilizar a página web e especificar o layout. O CSS descreve como os elementos HTML devem ser exibidos.

Design

É o que dá palatividade ao projeto. Existem técnicas bem definidas que vai desde a responsividade de layouts, passando pelo conteúdo até a melhor cor a ser aplicada nas páginas.

JavaScript

O JavaScript cuida das interações com o usuário e programa o comportamento das páginas da web.

Wordpress

Você também pode quer usar o Wordpress para criar seu site de conteúdo ou Loja Virtual com o Woocommerce. Se este for o caso posso construir seu site ou auxiliá-lo.
